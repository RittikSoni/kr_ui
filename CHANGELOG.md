## [0.0.1] - 2025-02-05

First public release of kr_ui - Premium glassmorphic UI components for Flutter!

### Added

#### Components
- **GlassyCard** - Beautiful glassmorphic card with customizable frosted glass effects
  - Full customization: blur, opacity, color, border, shadows
  - 5 presets: subtle, standard, strong, dark, colored
  - Optimized performance with efficient rendering
  
- **GlassyButton** - Interactive glassmorphic button with haptic feedback
  - Haptic feedback support (light/medium/heavy)
  - Press state animations
  - Full customization options
  - 5 presets: primary, secondary, danger, success, icon

#### Features
- Zero external dependencies (Pure Flutter)
- Comprehensive documentation with examples
- iOS-style design language
- Cross-platform support (iOS, Android, Web, Desktop)
- TypeScript-like parameter assertions for safety
- Preset configurations for quick implementation

#### Documentation
- Complete API documentation
- Usage examples for all components
- Contributing guide
- README with quick start guide

#### Infrastructure
- Melos monorepo setup
- Showcase app with Apple-style design
- Testing infrastructure
- CI/CD ready

### Performance
- Optimized BackdropFilter usage
- Efficient widget rebuilds
- 60fps animations
